The graphics are inspired by space simulators such as Celestia and Universe Sandbox while remaining optimized for mobile devices.

Features:
⭐ Real stars
🪐 Real planets
🌍 Detailed planetary data
🌌 Two explorable galaxies
🚀 Space exploration and simulation
